⚠ 级数与积分的桥梁

级数 $\sum a_n$ 的敛散性与积分 $\int f(x)\,dx$ 高度对应（积分判别法）。反常积分中的 $p$-积分标尺、Dirichlet/Abel 判别法在这里都有对应的离散版本。掌握一套，等于掌握两套。

第一部分：基本概念与柯西准则

定义 — 级数与部分和

给定数列 $(a_n)_{n \ge 1}$，定义部分和 $S_N = \sum_{n=1}^{N} a_n$。

级数 $\sum_{n=1}^{\infty} a_n$ 收敛，当且仅当 $\lim_{N \to \infty} S_N$ 存在且有限。此极限称为级数的和。

定理 — 柯西收敛准则（级数版）

$\sum a_n$ 收敛 $\iff$ $\forall \varepsilon > 0$，$\exists N_0$ 使得 $\forall m > n \ge N_0$：

$$\left|\sum_{k=n+1}^{m} a_k\right| < \varepsilon$$

直觉：级数的「尾部片段」可以任意小。

证明
证明：

$\sum a_n$ 收敛 $\iff$ $(S_N)$ 收敛 $\iff$ $(S_N)$ 是柯西列 $\iff$

$$\forall \varepsilon > 0,\ \exists N_0,\ \forall m > n \ge N_0: |S_m - S_n| < \varepsilon$$

而 $S_m - S_n = \sum_{k=n+1}^{m} a_k$，命题即得。

$\blacksquare$

第二部分：发散的必要条件

定理 — 通项趋于零是收敛的必要条件

若 $\sum a_n$ 收敛，则 $\lim_{n \to \infty} a_n = 0$。

证明
证明：

柯西准则取 $m = n+1$：$|a_{n+1}| = |S_{n+1} - S_n| < \varepsilon$（$n$ 充分大时）。故 $a_n \to 0$。

$\blacksquare$

💀 逆命题不成立！

$a_n \to 0$ 不能保证收敛。经典反例：调和级数 $\sum 1/n$，通项趋于 0 但发散。

实战价值： 逆否命题是秒杀利器——若 $a_n \not\to 0$，级数必发散。如 $\sum \cos(1/n)$：$\cos(1/n) \to 1 \neq 0$，直接判发散。

第三部分：正项级数 — $p$-级数与积分判别法

定理 — $p$-级数标尺 (Séries de Riemann)
$$\sum_{n=1}^{\infty} \frac{1}{n^p} \quad \text{当且仅当 } p > 1 \text{ 时收敛}$$

与无穷积分 $\int_1^{+\infty} 1/x^p\,dx$ 的判据完全一致。

定理 — 积分判别法 (Critère intégral)

设 $f : [1, +\infty) \to [0, +\infty)$ 连续且单调递减。则：

$$\sum_{n=1}^{\infty} f(n) \text{ 收敛} \iff \int_1^{+\infty} f(x)\,dx \text{ 收敛}$$

更精确地，有夹逼：$\int_1^{N+1} f \le \sum_{n=1}^N f(n) \le f(1) + \int_1^N f$。

积分判别法的证明
证明：

因 $f$ 单调递减，在 $[n, n+1]$ 上：$f(n+1) \le f(x) \le f(n)$。积分：

$$f(n+1) \le \int_n^{n+1} f(x)\,dx \le f(n)$$

对 $n = 1, 2, \ldots, N$ 求和：

$$\sum_{n=2}^{N+1} f(n) \le \int_1^{N+1} f(x)\,dx \le \sum_{n=1}^{N} f(n)$$

左侧 $= S_{N+1} - f(1)$。所以：

$$S_{N+1} - f(1) \le \int_1^{N+1} f \le S_N$$

若 $\int f$ 收敛（设极限为 $I$），则 $S_N \ge \int_1^{N+1} f \to I$（$S_N$ 单调递增有下界），且 $S_N \le f(1) + I$（有上界），故 $S_N$ 收敛。

若 $\int f$ 发散，则 $S_{N+1} \ge f(1) + \int_1^{N+1} f \to +\infty$，$\sum$ 发散。

$\blacksquare$

积分判别法的威力 — 证明调和级数发散

$f(x) = 1/x$，$\int_1^{+\infty} dx/x = +\infty$。由积分判别法，$\sum 1/n$ 发散。

同理可处理 Bertrand 级数 $\sum 1/(n(\ln n)^q)$：换元后化为积分判别法。

第四部分：d'Alembert 与 Cauchy 判别法

1. d'Alembert 比值法 (Critère de d'Alembert)

定理

设 $a_n > 0$，$\displaystyle l = \lim_{n \to \infty} \frac{a_{n+1}}{a_n}$。

$l$ 的值结论
$l < 1$$\sum a_n$ 收敛
$l > 1$$\sum a_n$ 发散
$l = 1$无法判断！

适用场景： 含 $n!$（阶乘）或 $a^n$（指数）的级数。

d'Alembert 判别法的证明
证明（$l < 1$ 的情形）：

取 $r$ 使 $l < r < 1$。由极限定义，$\exists N_0$ 使得 $n \ge N_0$ 时 $a_{n+1}/a_n \le r$。

递推：$a_{N_0 + k} \le r^k \cdot a_{N_0}$。

即 $a_n \le C \cdot r^n$（$n$ 充分大时），而 $\sum r^n$ 是公比 $r < 1$ 的几何级数，收敛。由比较判别法，$\sum a_n$ 收敛。

$l > 1$ 的情形： 同理 $a_{n+1}/a_n \ge 1$（$n$ 大时），$a_n$ 不趋于 0，由必要条件发散。

$\blacksquare$

2. Cauchy 根值法 (Critère de Cauchy / de la racine)

定理

设 $a_n \ge 0$，$\displaystyle l = \lim_{n \to \infty} \sqrt[n]{a_n}$（或 $\limsup$）。

  • $l < 1$：收敛。
  • $l > 1$：发散。
  • $l = 1$：无法判断。

适用场景： 整体带 $n$ 次方的级数，如 $\left(\frac{n}{n+1}\right)^{n^2}$。

Cauchy 根值法的证明
证明（$l < 1$）：

取 $r$ 使 $l < r < 1$。$\exists N_0$ 使得 $n \ge N_0$ 时 $\sqrt[n]{a_n} \le r$，即 $a_n \le r^n$。由 $\sum r^n$ 收敛和比较判别法，$\sum a_n$ 收敛。

$\blacksquare$

⚠ 根值法严格强于比值法

若比值法极限 $l$ 存在，则根值法极限也存在且相等。但根值法极限可能存在而比值法极限不存在。

例： $a_n = 2^{-n}$（$n$ 奇）、$a_n = 3^{-n}$（$n$ 偶）。比值 $a_{n+1}/a_n$ 在 $2/3$ 和 $3/2$ 之间交替，极限不存在。但 $\sqrt[n]{a_n} \to 1/\sqrt{6} < 1$（存在），根值法可判收敛。

💀 $l = 1$ 时两法均失效

$\sum 1/n$（发散）和 $\sum 1/n^2$（收敛）的比值极限和根值极限都是 $1$。遇到 $l = 1$，必须换用比较判别法、Raabe 判别法或凝聚判别法。

第五部分：Raabe 判别法

定理 — Raabe 判别法 (Critère de Raabe-Duhamel)

设 $a_n > 0$，$\displaystyle R = \lim_{n \to \infty} n\left(\frac{a_n}{a_{n+1}} - 1\right)$。

  • $R > 1$：$\sum a_n$ 收敛。
  • $R < 1$：$\sum a_n$ 发散。
  • $R = 1$：无法判断。

使用时机： d'Alembert 比值法给出 $l = 1$ 时的后续武器。

Raabe 判别法的证明思路
证明思路（$R > 1$）：

取 $\alpha$ 使 $1 < \alpha < R$。当 $n$ 充分大时 $n(a_n/a_{n+1} - 1) \ge \alpha$，即：

$$\frac{a_{n+1}}{a_n} \le \frac{n}{n + \alpha} = \frac{1}{1 + \alpha/n}$$

与 $b_n = 1/n^\alpha$ 比较：$b_{n+1}/b_n = (n/(n+1))^\alpha$。可以证明 $a_{n+1}/a_n \le b_{n+1}/b_n$（$n$ 充分大时），由此 $a_n \le C \cdot b_n = C/n^\alpha$。$\alpha > 1$ 故 $\sum b_n$ 收敛，由比较判别法 $\sum a_n$ 收敛。

$\blacksquare$

例 — Raabe 判别法实战

判断 $\sum_{n=1}^{\infty} \frac{(2n)!}{4^n (n!)^2}$ 的敛散性。

比值：$\frac{a_{n+1}}{a_n} = \frac{(2n+2)!}{4^{n+1}((n+1)!)^2} \cdot \frac{4^n (n!)^2}{(2n)!} = \frac{(2n+1)(2n+2)}{4(n+1)^2} = \frac{2n+1}{2(n+1)} \to 1$。

d'Alembert 失效！用 Raabe：

$$n\left(\frac{a_n}{a_{n+1}} - 1\right) = n\left(\frac{2(n+1)}{2n+1} - 1\right) = n \cdot \frac{1}{2n+1} = \frac{n}{2n+1} \to \frac{1}{2} < 1$$

Raabe 判定发散。（这就是中心二项式系数 $\binom{2n}{n}/4^n \sim 1/\sqrt{\pi n}$，是 $p = 1/2$ 级的。）

第六部分：Cauchy 凝聚判别法

定理 — Cauchy 凝聚判别法 (Critère de condensation)

设 $(a_n)$ 单调递减且 $a_n \ge 0$。则：

$$\sum_{n=1}^{\infty} a_n \text{ 收敛} \iff \sum_{k=0}^{\infty} 2^k a_{2^k} \text{ 收敛}$$
证明
证明：

将 $\sum a_n$ 按 $2$ 的幂次分组：第 $k$ 组包含 $a_{2^k}, a_{2^k+1}, \ldots, a_{2^{k+1}-1}$，共 $2^k$ 项。

因 $a_n$ 递减：

$$2^k a_{2^{k+1}} \le \sum_{n=2^k}^{2^{k+1}-1} a_n \le 2^k a_{2^k}$$

对 $k$ 求和：

$$\frac{1}{2}\sum_{k=0}^{K} 2^k a_{2^k} \le \sum_{n=1}^{2^{K+1}-1} a_n \le \sum_{k=0}^{K} 2^k a_{2^k}$$

两侧同敛散，命题得证。

$\blacksquare$

应用 — 一行证明 Bertrand 级数

$\sum 1/(n(\ln n)^q)$：凝聚后 $2^k a_{2^k} = 2^k / (2^k (k\ln 2)^q) = C/k^q$。

这是 $p$-级数！$q > 1$ 收敛，$q \le 1$ 发散。一步到位。

第七部分：交错级数与 Leibniz 准则

定理 — Leibniz 交错级数准则

对 $\sum_{n=1}^{\infty} (-1)^n u_n$（$u_n > 0$），若满足：

  1. $(u_n)$ 单调递减：$u_{n+1} \le u_n$。
  2. $\lim_{n \to \infty} u_n = 0$。

则级数收敛。且余项估计：$|R_N| = |\sum_{n=N+1}^{\infty} (-1)^n u_n| \le u_{N+1}$。

Leibniz 准则的证明
证明：

考察偶数部分和 $S_{2N}$：

$$S_{2N} = (u_1 - u_2) + (u_3 - u_4) + \cdots + (u_{2N-1} - u_{2N})$$

每个括号 $\ge 0$（因 $u_n$ 递减），故 $S_{2N}$ 单调递增。

另一方面：

$$S_{2N} = u_1 - (u_2 - u_3) - (u_4 - u_5) - \cdots - u_{2N} \le u_1$$

单调递增且有上界 $\implies$ $S_{2N} \to S$（某有限值）。

$S_{2N+1} = S_{2N} + u_{2N+1} \to S + 0 = S$。

所以 $S_N \to S$，级数收敛。

余项估计： $R_N$ 本身也是满足 Leibniz 条件的交错级数，其绝对值 $\le$ 首项 $u_{N+1}$。

$\blacksquare$

⚠ Leibniz 只能证收敛，不能证绝对收敛

如果 $\sum |a_n| = \sum u_n$ 也收敛，那是绝对收敛（更强）。Leibniz 通常用于证明条件收敛。

第八部分：Abel/Dirichlet 判别法（级数版）

定理 — Abel 变换（分部求和 / Sommation d'Abel）

设 $B_n = \sum_{k=1}^{n} b_k$（部分和）。则：

$$\sum_{k=1}^{N} a_k b_k = a_N B_N + \sum_{k=1}^{N-1}(a_k - a_{k+1}) B_k$$

这是分部积分在离散情形下的对应物，是 Dirichlet/Abel 判别法的核心引理。

定理 — Dirichlet 判别法（级数版）

若：(1) $(B_n)$ 有界；(2) $(a_n)$ 单调趋于 0。则 $\sum a_n b_n$ 收敛。

定理 — Abel 判别法（级数版）

若：(1) $\sum b_n$ 收敛；(2) $(a_n)$ 单调有界。则 $\sum a_n b_n$ 收敛。

应用 — Leibniz 是 Dirichlet 的特例

交错级数 $\sum (-1)^n u_n$：取 $b_n = (-1)^n$，$a_n = u_n$。$B_n$ 在 $0$ 和 $1$ 之间交替，有界；$u_n$ 单调趋于 0。满足 Dirichlet 条件。

第九部分：绝对收敛与条件收敛

定义
  • 绝对收敛： $\sum |a_n|$ 收敛。
  • 条件收敛： $\sum a_n$ 收敛但 $\sum |a_n|$ 发散。
定理 — 绝对收敛 $\implies$ 收敛

证明同积分情形：三角不等式 + 柯西准则。

Riemann 重排定理（超纲但极其深刻）

若 $\sum a_n$ 条件收敛，则对任意 $S \in \mathbb{R} \cup \{\pm\infty\}$，都存在 $\mathbb{N}$ 的一个排列 $\sigma$ 使得 $\sum a_{\sigma(n)} = S$。

换言之，条件收敛级数的和取决于求和顺序！只有绝对收敛级数才能任意重排不改变和。这是级数理论中最震撼的结果之一。

第十部分：经典反例与致命陷阱

💀 陷阱 1 — 交错项不等于收敛

$\sum (-1)^n \frac{n}{n+1}$：通项 $|a_n| = n/(n+1) \to 1 \neq 0$。必要条件不满足，直接发散！别看到 $(-1)^n$ 就条件反射说收敛。

💀 陷阱 2 — 等价无穷小用于级数的限制

$\sin(1/n) \sim 1/n$，所以 $\sum \sin(1/n)$ 与 $\sum 1/n$ 同敛散——发散。✓

但 $\sum (-1)^n \sin(1/n)$：不能用等价无穷小代替后说 $\sum (-1)^n/n$ 收敛所以原级数也收敛。正确做法：验证 $\sin(1/n)$ 单调递减趋于 0，Leibniz 适用，收敛。✓

规则： 等价无穷小只能用于正项级数的比较。变号级数中不能直接替换通项。

💀 陷阱 3 — 比值法不适用于 $p$-级数

$\sum 1/n^p$：$a_{n+1}/a_n = (n/(n+1))^p \to 1$。比值法失效。必须用比较判别法或积分判别法。

💀 陷阱 4 — 部分和有界不等于收敛

$a_n = (-1)^n$：部分和 $S_N$ 在 $-1$ 和 $0$ 之间交替，有界。但 $a_n \not\to 0$，级数发散。有界 $\neq$ 收敛！

第十一部分：高阶拓展（超纲）

1. Euler-Mascheroni 常数 $\gamma$

定义与存在性
$$\gamma = \lim_{N \to \infty} \left(\sum_{n=1}^{N} \frac{1}{n} - \ln N\right) \approx 0.5772$$

虽然调和级数 $\sum 1/n$ 发散，但它与 $\ln N$ 之间的差收敛到一个有限常数。

$\gamma$ 存在性的证明
证明：

设 $\gamma_N = \sum_{n=1}^{N} 1/n - \ln N$。

单调性： $\gamma_N - \gamma_{N+1} = -\frac{1}{N+1} + \ln(N+1) - \ln N = \ln\left(1 + \frac{1}{N}\right) - \frac{1}{N+1}$。

由不等式 $\ln(1+x) \ge x/(1+x)$（$x > 0$），取 $x = 1/N$：$\ln(1+1/N) \ge 1/(N+1)$。故 $\gamma_N \ge \gamma_{N+1}$，单调递减。

有下界： 积分判别法的估计给出 $\sum_{n=1}^{N} 1/n \ge \int_1^{N+1} dx/x = \ln(N+1)$，但更精确的分析给出 $\gamma_N \ge 0$。

单调递减有下界 $\implies$ 极限存在。

$\blacksquare$

$\gamma$ 的应用

调和级数的精确渐近：$H_N = \sum_{n=1}^N 1/n = \ln N + \gamma + O(1/N)$。

在概率论（Gumbel 分布）、数论（素数定理相关常数）中反复出现。

2. Stirling 公式

$n! \sim \sqrt{2\pi n} \left(\frac{n}{e}\right)^n$

精确含义：$\lim_{n \to \infty} \frac{n!}{\sqrt{2\pi n}(n/e)^n} = 1$。

实战价值： 遇到含 $n!$ 的级数但比值法 $l = 1$ 时，用 Stirling 估计通项的渐近量级，再用比较判别法。

例： $\frac{(2n)!}{4^n (n!)^2} \sim \frac{\sqrt{4\pi n}(2n/e)^{2n}}{4^n \cdot 2\pi n \cdot (n/e)^{2n}} = \frac{1}{\sqrt{\pi n}}$。这是 $1/n^{1/2}$ 级，发散。

3. Riemann $\zeta$ 函数初步

定义与特殊值
$$\zeta(s) = \sum_{n=1}^{\infty} \frac{1}{n^s} \qquad (s > 1 \text{ 时收敛})$$

特殊值（部分可用 Fourier 级数证明）：

  • $\zeta(2) = \pi^2/6$（Basel 问题，Euler 1735）
  • $\zeta(4) = \pi^4/90$
  • $\zeta(2k) = (-1)^{k+1} \frac{(2\pi)^{2k} B_{2k}}{2(2k)!}$（$B_{2k}$ 是 Bernoulli 数）

$\zeta(3) \approx 1.202$（Apéry 常数，已证无理性，但无闭合表达式）。$\zeta$ 函数在数论中具有核心地位（Riemann 猜想、素数分布）。

4. 级数求和技巧速查

常用求和工具
  • 望远镜级数（télescopage）： $\sum (b_n - b_{n+1}) = b_1 - \lim b_n$。
  • 部分分式分解： $\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$，化为望远镜。
  • 泰勒展开生成： $\sum x^n/n! = e^x - 1$，$\sum (-1)^n x^{2n}/(2n)! = \cos x - 1$。
  • Abel 求和 + 逐项求导/积分： 对幂级数 $\sum a_n x^n$ 操作后取 $x = 1$。如 $\sum 1/n^2$ 可由 $-\ln(1-x)/x$ 的积分推导。
